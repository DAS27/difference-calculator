## Addons

Use <http://psysh.org/>

### Installation

Use [Composer](https://getcomposer.org/)

```bash
$ composer global require damir/difference-calculator:dev-master
```

## Setup

```sh
$ git clone https://github.com/DAS27/difference-calculator.git

$ make install
```

## Run tests

```sh
$ make test
```
